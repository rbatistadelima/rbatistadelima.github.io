Legacy WebRTC Room

A small VDO.ninja-style peer-to-peer camera room with manual signaling and older Chrome fallbacks.

How to use: Host clicks Start camera → Create offer → sends text to Guest. Guest pastes it → Set remote text → Create answer → sends answer back. Host pastes answer → Set remote text.

Notes: This uses WebRTC. It can support older Chrome builds that still have WebRTC, but very old Chrome versions without RTCPeerConnection or secure getUserMedia will not work. For internet connections behind strict NAT, a TURN server may be needed.
